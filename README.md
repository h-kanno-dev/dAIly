# dAIly
## ■概要  
LINEで送った曖昧な時間をGeminiが即座に解析。  
カレンダー登録とリマインド設定をシームレスにつなぐ、次世代スケジュール管理ツールです。  
日時だけでなく、場所や詳細メモの登録もLINEで完結します。

## ■主な機能
- 自然言語による予定解析
- Googleカレンダー自動登録
- コンテキストに合わせたリマインド通知

## ■使い方
- 友だち追加
LINE公式アカウントを友だち追加します。

- 予定をメッセージで送信
「明日10時に会議」「来週の月曜19時から渋谷で飲み会」など、日常の言葉で送信します。

- 通知設定の選択
Botから確認が届くので、リママインドしたいタイミング（15分前、1時間前など）をボタンで選びます。
「カスタム」を選ぶと任意の通知時刻を設定することが可能です。

- 自動登録＆リマインド
Googleカレンダーに予定が登録され、指定した時間に通知をお届けします✨

## ■利用シーン
- 手が離せない時に、話し言葉でパッと予定を入れたい時
- カレンダー登録とリマインド設定を、一度の操作で終わらせたい時

## ■入力例
- 明日18時から渋谷で焼肉（場所の自動抽出）
- 20時からZoom会議 URLはメールを確認すること（詳細メモの登録）
- 15分後に洗濯物を取り出す（相対時間の解析）

## ■ポイント  
- **対話型UIによるストレスフリーな登録**
Botからの問いかけに対し、「明日の8時に教えて」と自然に答えるだけで通知時刻をセット。カレンダーの各項目を一つずつ手入力する手間を省き、「LINEで会話するように予定登録を完結させる」直感的なUIを実現しました。

- **クイックリプライによる入力補助**
通知時刻をワンタップで選べる「クイックリプライボタン」を導入。入力の心理的ハードルを下げ、伊弁性を追求しています。

- **AI（Gemini）による親しみやすさの演出**
Geminiにより、登録内容に合わせたパーソナライズされた返信を生成。単なる事務的なツールではなく、毎日使いたくなる「楽しさ」という付加価値をプラスしました。

- **「予定」と「通知」の切り分けロジック**
カレンダーの予定時刻とは別に、スプレッドシートをキューとして活用することで、ユーザーが本当に通知してほしいタイミングを自由に選べる柔軟な設計を行いました。

## ■システムフロー  
LINEメッセージ ➔ GAS ➔ Gemini API (解析) ➔ Googleカレンダー (登録) ＆ スプレッドシート (キュー蓄積) ➔ 時間指定トリガー ➔ LINEプッシュ通知

## ■技術構成:  
- 言語 / プラットフォーム：GAS（Google Apps Sprict）
- API：LINE Messaging API / Google Calendar API / Gemini API
- DB：Google Spreadsheets
