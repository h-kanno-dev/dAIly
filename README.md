# dAIly
■概要:
LINEで送った曖昧な時間をGeminiが即座に解析。
カレンダー登録とリマインド設定をシームレスにつなぐ、次世代スケジュール管理ツールです。

■主な機能:
・自然言語による予定解析
・Googleカレンダー自動登録
・コンテキストに合わせたリマインド通知

■ポイント: 
・**対話型UIによるストレスフリーな登録**
Botからの問いかけに対し、「明日の8時に教えて」と自然に答えるだけで通知時刻をセット。カレンダーの各項目を一つずつ手入力する手間を省き、「LINEで会話するように予定登録を完結させる」直感的なUIを実現しました。

・**クイックリプライによる入力補助**
通知時刻をワンタップで選べる「クイックリプライボタン」を導入。入力の心理的ハードルを下げ、伊弁性を追求しています。

・**AI（Gemini）による親しみやすさの演出**
Geminiにより、登録内容に合わせたパーソナライズされた返信を生成。単なる事務的なツールではなく、毎日使いたくなる「楽しさ」という付加価値をプラスしました。

■技術構成: 
- 言語 / プラットフォーム：GAS（Google Apps Sprict）
- API：LINE Messaging API / Google Calendar API / Gemini API
- DB：Google Spreadsheets
