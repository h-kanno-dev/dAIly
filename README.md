# dAIly
## ■概要  
LINEで送った曖昧な時間をGeminiが即座に解析。  
カレンダー登録とリマインド設定をシームレスにつなぐ、次世代スケジュール管理ツールです。  
日時だけでなく、場所や詳細メモの登録もLINEで完結します。

## ■主な機能
- 自然言語による予定解析
- Googleカレンダー自動登録
- コンテキストに合わせたリマインド通知

## ■システムフロー  
LINEメッセージ ➔ GAS ➔ Gemini API (解析) ➔ Googleカレンダー (登録) ＆ スプレッドシート (キュー蓄積) ➔ 時間指定トリガー ➔ LINEプッシュ通知

## ■使い方
- 友だち追加  
LINE公式アカウントを友だち追加します。

- 予定をメッセージで送信  
「明日10時に会議」「来週の月曜19時から渋谷で飲み会」など、日常の言葉で送信します。

- 通知設定の選択  
Botから確認が届くので、リママインドしたいタイミング（15分前、1時間前など）をボタンで選びます。
「カスタム」を選ぶと任意の通知時刻を設定することが可能です。

- 自動登録＆リマインド  
Googleカレンダーに予定が登録され、指定した時間に通知をお届けします✨

## ■利用シーン
- **カレンダーアプリを開く手間を最小化**
項目を一つずつ手入力する煩わしさを解消し、メッセージを送るだけで登録を完了させたいとき
- **予定登録とリマインド設定の一元化**
複数のステップが必要なスケジュール管理を、一度の操作でスマートに完結させたいとき

## ■入力例
- 明日18時から渋谷で焼肉（場所の自動抽出）
- 20時からZoom会議 URLはメールを確認すること（詳細メモの登録）
- 15分後に洗濯物を取り出す（相対時間の解析）

## ■ポイント  
- **対話型UIによるストレスフリーな登録**
単なるコマンド入力ではなく、Botからの問いかけに答えるだけで完了する対話形式を採用。カレンダーの各項目を一つずつ手入力する手間を省き、「LINEで会話するように予定登録を完結させる」直感的なUIを実現しました。

- **「予定」と「通知」の切り分けロジック**
カレンダーの予定時刻とは別に、スプレッドシートをキューとして活用することで、ユーザーが本当に通知してほしいタイミングを自由に選べる柔軟な設計を行いました。

- **独自ロギングシステムの構築**
スプレッドシートに専用の「ログ出力シート」を作成。エラー内容や変数の中身をリアルタイムで書き出す独自ロギング機能を構築しました。エラー箇所が可視化されたことで開発効率が大幅に向上。AIへの正確な情報提供も可能になり、迅速な機能改善サイクルを実現しました。
