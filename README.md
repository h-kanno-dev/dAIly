# dAIly（デイリー）

## ■システムフロー
LINEユーザー  
↓  
LINE Messaging API  
↓  
GAS Webhook  
↓  
Gemini API（自然言語解析）  
↓  
Googleカレンダー（予定登録）  
+  
Googleスプレッドシート（リマインドキュー）  
↓  
時間トリガー  
↓  
LINE Push通知

## ■概要  
LINEで送った曖昧な予定をGeminiが解析し、Googleカレンダー登録とリマインド通知を自動化するケジュール管理Botです。  
カレンダーアプリを開いて日時・タイトル・通知設定を個別に入力する手間を減らし、「思いついた瞬間にLINEで送るだけ」で予定管理を完結させることを目的として開発しました。

## ■開発の背景
日々の仕事や生活の中で、ふとした瞬間に発生する予定を、その場ですぐに記録できないもどかしさがありました。  
後で落ち着いてからカレンダーを開こうと思っても、忙しさに追われるうちに詳細を忘れてしまったり、正確な登録が難しくなったりすることが課題でした。  
そこで、日常的に使い慣れているLINEを通じて、思いついたその時に「言葉」として送るだけで、確実にスケジュールを残せる仕組みを目指しました。  

## ■主な機能
- **自然言語による予定解析**：Gemini APIを活用し、日常的な話し言葉から日時や場所を抽出。
- **Googleカレンダー自動登録**：解析した予定を、Calendar APIを通じて指定のカレンダーへ即座に反映。
- **柔軟なリマインド設定**：スプレッドシートをキューとして活用し、ユーザーの生活リズムに合わせた通知を実現。

## ■使い方
- **友だち追加**  
LINE公式アカウントを友だち追加します。

- **予定をメッセージで送信**  
「明日10時に会議」「来週の月曜19時から渋谷で飲み会」など、日常の言葉で送信します。

- **通知設定の選択**  
Botから確認が届くので、リマインドしたいタイミング（15分前、1時間前など）をボタンで選びます。  
「カスタム」を選ぶと任意の通知時刻を設定することが可能です。

- **自動登録＆リマインド**  
Googleカレンダーに予定が登録され、指定した時間に通知をお届けします✨

## ■入力例
- 明日18時から渋谷で焼肉（場所の自動抽出）
- 20時からZoom会議 URLはメールを確認すること（詳細メモの登録）
- 15分後に洗濯物を取り出す（相対時間の解析）

## ■ポイント  
- **対話型UIによるストレスフリーな登録**  
単なるコマンド入力ではなく、Botからの問いかけに答えるだけで完了する対話形式を採用。カレンダーの各項目を一つずつ手入力する手間を省き、「LINEで会話するように予定登録を完結させる」直感的なUIを実現しました。

- **自由度の高いリマインドタイミング設定**  
カレンダーの予定時刻に縛られず、スプレッドシートをキューとして活用することで、「15分前」「1日前」あるいは「特定のカスタム時刻」など、ユーザーが本当に通知を欲するタイミングを柔軟に指定できる設計にこだわりました。

- **独自ロギングシステムの構築**  
  スプレッドシートを活用したリアルタイムログ機能を実装。エラー発生時の状況を正確に記録することで、外部APIとの連携トラブルにも迅速に対応可能な、保守性の高い運用体制を整えました。
